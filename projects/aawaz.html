Aawaz

  • Tech Stack: HTML, CSS, JavaScript, PHP, API Integration

Overview: Aawaz is a multilingual translation engine designed to make railway station announcements accessible in five languages, serving over 1.5 million daily passengers.

Key Contributions:

  • Developed the frontend using HTML, CSS, and JavaScript, ensuring a responsive and user-friendly interface.
  • Integrated APIs for real-time translation and announcement scheduling, enhancing system performance.
  • Implemented dynamic language switching to support accessibility and inclusivity.
  • Streamlined backend operations with PHP for efficient data handling and API requests.

Impact: The project improved communication for passengers, making travel seamless for non-native speakers and ensuring critical announcements reached a diverse audience.

Key Features:

  • Multilingual Announcement Translation
  • Real-Time Scheduling and Notifications
  • User-Friendly Admin Dashboard
  • APIs for Enhanced Functionality
